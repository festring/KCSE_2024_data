# coupang_review_dataset
본 데이터셋은 KCSE_2024에 게재된 "KNN 기반의 가짜 리뷰 계정 분류 모델 및 서비스 개발" 논문에 사용된 데이터 입니다. 

## Basic Information
 
|클래스|Info|
|:----:|:------:|
|리뷰수 | 해당 계정이 작성한 리뷰개수  | 
|성실도  | 전체 리뷰 중 ‘성실한 리뷰’ 비율    | 
|쿠팡비율  |전체 리뷰 중 판매자가 쿠팡(주)로 표기된 제품에 작성한 리뷰의 비율|
|글자수 |평균 리뷰 글자수      |
|작성일수 | 리뷰를 작성된 날짜의 수 |
|알바여부 |해당 계정의 알바 여부입니다. 0은 일반 계정, 1은 가짜 계정입니다. |
